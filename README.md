# Esse é um projeto de estudos (:
Esse é o projeto final do curso do [JSNinja](https://queroser.ninja/promocoes) 

Ele foi começado durante o curso acima, e pretendo usá-lo como playground de testes e estudos.

Pretendo deixar esse código menos horroroso adotando boas de ES6
E depois aplicar os conhecimentos de organização de CSS pra que ele fique visualmente bacana também (:

## O que o projeto faz?
Ele é um formulário de cadastro de carros.
Você preenche alguns dados - como imagem, ano, placa, cor, modelo e etc - e ele renderiza essas informações na tela
(e é isso :P)
